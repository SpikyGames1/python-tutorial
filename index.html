1. print()

The print() function displays text or values on the screen. It's how your program talks to you!

Pythonprint("Hello, World!")
print("I'm learning Python!")
print(42)
print(3.14)
üí° Tip: Text (strings) must be wrapped in quotes "like this" or 'like this'. Numbers don't need quotes.

2. Variables

Variables are like labeled boxes that store information. You can put data in them and use them later!

Python# Creating variables
name = "Alex"
age = 16
height = 5.7
is_student = True

# Using variables
print(name)
print("I am", age, "years old")

Variable Rules:

  • Names can contain letters, numbers, and underscores
  • Must start with a letter or underscore (not a number)
  • Case-sensitive: Name and name are different
  • Use descriptive names: student_count is better than sc

3. Type Conversion

Converting data from one type to another. Sometimes you need a number as text, or text as a number!

Implicit Conversion (Automatic)

Python does this automatically when it's safe:

Pythonx = 5        # integer
y = 2.5      # float
result = x + y    # Python converts x to float automatically
print(result)  # Output: 7.5

Explicit Conversion (Manual)

You tell Python exactly what to convert:

Python# String to Integer
age_text = "25"
age_number = int(age_text)
print(age_number + 5)  # Output: 30

# Integer to String
score = 100
message = "Your score is " + str(score)
print(message)  # Output: Your score is 100

# String to Float
price = float("19.99")
print(price)  # Output: 19.99
üí° Common Functions: int() ‚Üí whole numbers, float() ‚Üí decimals, str() ‚Üí text

4. Operators

Operators let you perform calculations and comparisons.

Arithmetic Operators

Operator Name Example Result
+ Addition 5 + 3 8
- Subtraction 5 - 3 2
* Multiplication 5 * 3 15
/ Division 5 / 2 2.5
// Floor Division 5 // 2 2 (no decimals)

Comparison Operators

Pythona = 10
b = 5

print(a == b)  # Equal to: False
print(a != b)  # Not equal: True
print(a > b)   # Greater than: True
print(a < b)   # Less than: False
print(a >= b)  # Greater or equal: True
print(a <= b)  # Less or equal: False

Logical Operators

Pythonx = True
y = False

print(x and y)  # Both must be True: False
print(x or y)   # At least one True: True
print(not x)     # Opposite: False

5. Lists

Lists are collections that can hold multiple items. Think of them as a container with numbered slots!

Python# Creating a list
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, True, 3.14]

# Accessing items (index starts at 0!)
print(fruits[0])  # Output: apple
print(fruits[1])  # Output: banana
print(fruits[-1]) # Output: cherry (last item)

Common List Methods

Pythonmy_list = [1, 2, 3]

# append() - add to the end
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]

# insert() - add at specific position
my_list.insert(1, 99)
print(my_list)  # [1, 99, 2, 3, 4]

# remove() - remove first occurrence of value
my_list.remove(99)
print(my_list)  # [1, 2, 3, 4]

# pop() - remove and return last item (or at index)
last = my_list.pop()
print(last)     # 4
print(my_list)  # [1, 2, 3]

# index() - find position of value
position = my_list.index(2)
print(position)  # 1

# count() - count occurrences
numbers = [1, 2, 2, 3, 2]
print(numbers.count(2))  # 3

# sort() - sort the list
numbers = [3, 1, 4, 1, 5]
numbers.sort()
print(numbers)  # [1, 1, 3, 4, 5]

# reverse() - reverse the list
numbers.reverse()
print(numbers)  # [5, 4, 3, 1, 1]

# clear() - remove all items
numbers.clear()
print(numbers)  # []

# len() - get list length
fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # 3
üí° Remember: Lists are mutable (changeable) and use square brackets [ ]

6. Tuples

Tuples are like lists, but immutable (can't be changed after creation). Perfect for data that shouldn't be modified!

Python# Creating a tuple
coordinates = (10, 20)
rgb_color = (255, 128, 0)
person = ("Alice", 25, "Engineer")

# Accessing items (same as lists)
print(coordinates[0])  # Output: 10
print(person[1])       # Output: 25

# This will cause an error (tuples can't be changed):
# coordinates[0] = 15  ‚ùå TypeError!

When to use Tuples vs Lists:

  • Tuples: Use when data shouldn't change (like coordinates, RGB values, dates)
  • Lists: Use when you need to add/remove/modify items
üí° Remember: Tuples use parentheses ( ) and are immutable (can't be changed)

7. input()

The input() function lets your program ask questions and get answers from the user!

Python# Getting user input
name = input("What's your name? ")
print("Hello,", name)

# Important: input() always returns a string!
age_text = input("How old are you? ")
age = int(age_text)  # Convert to number
print("In 5 years, you'll be", age + 5)

# Shorter version (convert directly)
height = float(input("Enter your height in meters: "))
print("Your height is", height, "meters")
üí° Important: input() always gives you a string, even if the user types a number. Use int() or float() to convert when needed!
