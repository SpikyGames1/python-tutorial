1. print()

The print() function displays text or values on the screen. It's how your program talks to you!

Pythonprint("Hello, World!")
print("I'm learning Python!")
print(42)
print(3.14)
üí° Tip: Text (strings) must be wrapped in quotes "like this" or 'like this'. Numbers don't need quotes.

2. Variables

Variables are like labeled boxes that store information. You can put data in them and use them later!

Python# Creating variables
name = "Alex"
age = 16
height = 5.7
is_student = True

# Using variables
print(name)
print("I am", age, "years old")

Variable Rules:

  • Names can contain letters, numbers, and underscores
  • Must start with a letter or underscore (not a number)
  • Case-sensitive: Name and name are different
  • Use descriptive names: student_count is better than sc

3. Type Conversion

Converting data from one type to another. Sometimes you need a number as text, or text as a number!

Implicit Conversion (Automatic)

Python does this automatically when it's safe:

Pythonx = 5        # integer
y = 2.5      # float
result = x + y    # Python converts x to float automatically
print(result)  # Output: 7.5

Explicit Conversion (Manual)

You tell Python exactly what to convert:

Python# String to Integer
age_text = "25"
age_number = int(age_text)
print(age_number + 5)  # Output: 30

# Integer to String
score = 100
message = "Your score is " + str(score)
print(message)  # Output: Your score is 100

# String to Float
price = float("19.99")
print(price)  # Output: 19.99
üí° Common Functions: int() ‚Üí whole numbers, float() ‚Üí decimals, str() ‚Üí text

5. Lists

Lists are collections that can hold multiple items. Think of them as a container with numbered slots!

Python# Creating a list
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, True, 3.14]

# Accessing items (index starts at 0!)
print(fruits[0])  # Output: apple
print(fruits[1])  # Output: banana
print(fruits[-1]) # Output: cherry (last item)

Common List Methods

Pythonmy_list = [1, 2, 3]

# append() - add to the end
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]

# insert() - add at specific position
my_list.insert(1, 99)
print(my_list)  # [1, 99, 2, 3, 4]

# remove() - remove first occurrence of value
my_list.remove(99)
print(my_list)  # [1, 2, 3, 4]

# pop() - remove and return last item (or at index)
last = my_list.pop()
print(last)     # 4
print(my_list)  # [1, 2, 3]

# index() - find position of value
position = my_list.index(2)
print(position)  # 1

# count() - count occurrences
numbers = [1, 2, 2, 3, 2]
print(numbers.count(2))  # 3

# sort() - sort the list
numbers = [3, 1, 4, 1, 5]
numbers.sort()
print(numbers)  # [1, 1, 3, 4, 5]

# reverse() - reverse the list
numbers.reverse()
print(numbers)  # [5, 4, 3, 1, 1]

# clear() - remove all items
numbers.clear()
print(numbers)  # []

# len() - get list length
fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # 3
üí° Remember: Lists are mutable (changeable) and use square brackets [ ]

6. Tuples

Tuples are like lists, but immutable (can't be changed after creation). Perfect for data that shouldn't be modified!

Python# Creating a tuple
coordinates = (10, 20)
rgb_color = (255, 128, 0)
person = ("Alice", 25, "Engineer")

# Accessing items (same as lists)
print(coordinates[0])  # Output: 10
print(person[1])       # Output: 25

# This will cause an error (tuples can't be changed):
# coordinates[0] = 15  ‚ùå TypeError!

When to use Tuples vs Lists:

  • Tuples: Use when data shouldn't change (like coordinates, RGB values, dates)
  • Lists: Use when you need to add/remove/modify items
üí° Remember: Tuples use parentheses ( ) and are immutable (can't be changed)

7. input()

The input() function lets your program ask questions and get answers from the user!

Python# Getting user input
name = input("What's your name? ")
print("Hello,", name)

# Important: input() always returns a string!
age_text = input("How old are you? ")
age = int(age_text)  # Convert to number
print("In 5 years, you'll be", age + 5)

# Shorter version (convert directly)
height = float(input("Enter your height in meters: "))
print("Your height is", height, "meters")
üí° Important: input() always gives you a string, even if the user types a number. Use int() or float() to convert when needed!

8. if Statements

Make decisions in your code! if statements let your program do different things based on conditions.

Python# Basic if statement
age = 18
if age >= 18:
    print("You can vote!")

# if-else
temperature = 25
if temperature > 30:
    print("It's hot outside!")
else:
    print("It's nice outside!")

# if-elif-else (multiple conditions)
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

Using Logical Operators with if:

Pythonage = 20
has_license = True

# and - both conditions must be True
if age >= 18 and has_license:
    print("You can drive!")

# or - at least one condition must be True
is_weekend = True
is_holiday = False
if is_weekend or is_holiday:
    print("Time to relax!")

# not - reverses the condition
is_raining = False
if not is_raining:
    print("Let's go outside!")
üí° Important: Indentation matters in Python! The code inside an if block must be indented (usually 4 spaces or 1 tab).

9. for Loops

for loops repeat code a specific number of times. Perfect for going through lists or doing something multiple times!

Looping through a list:

Pythonfruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry

Using range():

Python# Print numbers 0 to 4
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# Print numbers 1 to 5
for i in range(1, 6):
    print(i)  # 1, 2, 3, 4, 5

# Print even numbers from 0 to 10
for i in range(0, 11, 2):
    print(i)  # 0, 2, 4, 6, 8, 10

Practical Examples:

Python# Sum all numbers in a list
numbers = [1, 2, 3, 4, 5]
total = 0
for num in numbers:
    total = total + num
print("Total:", total)  # 15

# Print multiplication table
for i in range(1, 11):
    print(f"5 x {i} = {5 * i}")
üí° range() syntax: range(stop), range(start, stop), or range(start, stop, step)

10. while Loops

while loops keep repeating as long as a condition is True. Great when you don't know exactly how many times to loop!

Python# Basic while loop
count = 0
while count < 5:
    print(count)
    count = count + 1  # Don't forget to update!

# Output: 0, 1, 2, 3, 4

Using while with user input:

Python# Keep asking until correct password
password = ""
while password != "python123":
    password = input("Enter password: ")
print("Access granted!")

# Simple game - guess the number
secret = 7
guess = 0
while guess != secret:
    guess = int(input("Guess a number (1-10): "))
    if guess < secret:
        print("Too low!")
    elif guess > secret:
        print("Too high!")
print("You got it!")

break and continue:

Python# break - exit the loop immediately
while True:
    answer = input("Type 'quit' to exit: ")
    if answer == "quit":
        break
    print("You typed:", answer)

# continue - skip to next iteration
count = 0
while count < 5:
    count += 1
    if count == 3:
        continue  # Skip printing 3
    print(count)  # Prints: 1, 2, 4, 5
‚ö†Ô∏è Warning: Make sure your while loop condition will eventually become False, or it will run forever (infinite loop)! Always update your condition variable inside the loop.
